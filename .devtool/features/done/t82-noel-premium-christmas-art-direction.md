# t82-noel-premium-christmas-art-direction - Redesign Noel as a premium Christmas experience

## Scope

Redesign the Noel invitation theme using the user's previous Reverie introduction as inspiration without recreating it. Noel should feel unmistakably Christmas: festive, peaceful, cold, happy, and premium across desktop and mobile.

## Acceptance

- [x] Noel has a distinct winter-conservatory hero composition in light and dark modes.
- [x] Falling snow respects reduced-motion preferences.
- [x] Evergreen boughs, restrained berry/ribbon details, frost, and designed borders feel festive without clip-art clutter.
- [x] Hero facts, section cards, imagery, location, and RSVP share a coherent Noel-owned visual system.
- [x] The design remains responsive, readable, accessible, and compatible with every event type.
- [x] Repository, production-build, contrast, SVG-asset, responsive, and reduced-motion verification pass. In-app browser screenshot QA was attempted but the browser surface was unavailable in this session.
