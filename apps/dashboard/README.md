# Lumiere Dashboard App UI Notes

Use the root `SKILL.md` for UI decisions.

Design read: the dashboard is the event manager workspace. It should feel calm, precise, and operational, with dense but readable layouts for event setup, guest groups, responses, and activity.

Token notes:

- Neutral administrative palette with a restrained Lumiere accent.
- Tighter radius and denser spacing than the public invite.
- Management states should be scannable and never rely on color alone.
- Event theme styling belongs in previews, not in core controls.
